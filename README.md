# 🛡️ El Escudo del Autor

**Guía de Supervivencia Editorial** — Una SPA en React que ayuda a los autores a entender la realidad del marketing editorial y construir su propia estrategia de lanzamiento.

## ✨ Funcionalidades

- **Checklist Interactivo** — Con lógica de fechas (90/60/45/30 días antes del lanzamiento) y persistencia en `localStorage`
- **El Ecosistema de Ventas** — Gráfico animado (Framer Motion + SVG) que visualiza el esfuerzo editorial vs. el del autor
- **i18n ES/EN** — Cambio de idioma completo en tiempo real
- **Light/Dark Mode** — Con persistencia en `localStorage`
- **Bibliografía** — Referencias editoriales reales (Sara Roahen, HMD Publishing)

## 🛠️ Stack

- React 19 + Vite
- Framer Motion
- CSS Variables (sin framework externo)

## 🚀 Desarrollo

```bash
npm install
npm run dev
```
